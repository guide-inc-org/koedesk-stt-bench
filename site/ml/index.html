മുൻകൂട്ടി രജിസ്റ്റർ ചെയ്ത രീതിശാസ്ത്രത്തിന് കീഴിൽ അളന്ന ബഹുഭാഷാ സ്പീച്ച്-ടു-ടെക്സ്റ്റ് കൃത്യത — Track A (FLEURS), 13 ഹോസ്റ്റഡ് എഞ്ചിൻ വേരിയന്റുകൾ × 12 ഭാഷകൾ × 200 ഉച്ചാരണങ്ങൾ.

താൽപ്പര്യ വൈരുദ്ധ്യം, മുൻകൂട്ടി വ്യക്തമാക്കുന്നു: ElevenLabs Scribe v2 ഉപയോഗിക്കുന്ന ഡിക്റ്റേഷൻ ഉൽപ്പന്നമായ koedesk ആണ് ഈ ബെഞ്ച്മാർക്ക് നിർമ്മിച്ച് ധനസഹായം നൽകുന്നത്. ഞങ്ങൾ നിഷ്പക്ഷരാണെന്ന് ഭാവിക്കുന്നില്ല — പകരം, ഞങ്ങളെ വിശ്വസിക്കാതെ സ്വയം ബെഞ്ച്മാർക്ക് വീണ്ടും പ്രവർത്തിപ്പിക്കാൻ ആവശ്യമായതെല്ലാം പ്രസിദ്ധീകരിച്ചിട്ടുണ്ട്: രീതിശാസ്ത്രം (പ്രധാന റണ്ണുകൾക്ക് മുൻപ് ഉറപ്പിച്ചത് — commit ചരിത്രം കാണുക), ഓരോ അസംസ്കൃത API പ്രതികരണവും (results/raw), എല്ലാ സ്കോറിംഗ് കോഡും, ഏതൊക്കെ എഞ്ചിനുകൾ ഉൾപ്പെടുത്താമെന്ന് തീരുമാനിച്ച സേവന നിബന്ധനകളുടെ ഓഡിറ്റും. Scribe പരാജയപ്പെടുന്നിടത്ത്, ആ ഫലം മാറ്റമില്ലാതെ പ്രസിദ്ധീകരിക്കുന്നു.
Track B ഇനിയും വരാനിരിക്കുന്നു. എഞ്ചിനുകൾ പരിശീലനം നേടിയിരിക്കാൻ സാധ്യതയുള്ള പൊതു ടെസ്റ്റ് സെറ്റുകളാണ് Track A ഉപയോഗിക്കുന്നത്; ബെഞ്ച്മാർക്കിന്റെ പ്രധാന ചോദ്യം — പുതിയതും മലിനീകരണരഹിതവുമായ 2026 ഓഡിയോയിൽ റാങ്കിംഗ് നിലനിൽക്കുമോ? — എന്നതിന് Track B ഉത്തരം നൽകും, Track A ⇄ Track B വ്യത്യാസം പ്രാധാന്യത്തോടെ റിപ്പോർട്ട് ചെയ്ത് ഇവിടെ പ്രസിദ്ധീകരിക്കും (§9).
